# XTCinstaller

#### 介绍
此项目由ReX 组织下ReX Wear 项目的 ReX iMoo Team 团队合力开发，版权及著作权属于ReX群组

#### 软件架构
x86或x86_64的Windows10/11系统

#### 安装教程

1.  运行安装程序
2.  仔细阅读后同意条款
3.  等待安装程序释放程序主体
4.  安装程序在释放结束后将自动运行驱动安装程序

#### 使用说明

1.  请使用Windows10/11完整版系统
2.  请在使用四点线连接手表后运行此程序
3.  请勿修改，注入此程序，否则可能无法正常运行
